⚙️ Settings
Anthropic (required)
Create a key named "content-engine" at console.anthropic.com → API Keys.
Shopify store domain
Your myshopify.com domain. Find it in Shopify admin — it's in the browser URL as admin.shopify.com/store/YOUR-STORE or check Settings → Domains.
Buffer publishing (optional)
Get your Personal Key: Buffer → Settings → API → Personal Keys → + New Key. Name it anything, set expiry to 1 year. Copy the key and add it to Netlify → Site settings → Environment variables as BUFFER_TOKEN. Then tap Load channels below.
